# Открыть в браузере
http://localhost:3000/


### Запуск приложения

#### Соберите Docker-образ
```bash
npm run dev
```

#### Соберите Docker-образ
```bash
docker build -t sports_city .
```

#### Запустите Docker-контейнер
```bash
docker run -p 3000:3000 sports_city
```

#### Собрать контейнер API и запустить
```bash
docker-compose --env-file .env.local up --build
```
